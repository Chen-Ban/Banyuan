# Banyan 榕

> 独木成林，气根落地，生生不息。

**Banyan** 是基于 BanvasGL 引擎的低代码可视化设计平台。

取名"榕"，因榕树一木成森——气根自枝垂落，触地即为新干，百干同根，枝叶交织，化为一片自生自洽的微型生态。恰如此平台：以 BanvasGL 为根基，组件为气根，每一次拖拽落地皆成应用，万千形态共享同一脉络，和谐共生，美美与共。

## 设计理念

- **根深叶茂** — 底层引擎稳固，上层应用自由生长
- **一木成林** — 一套画布能力，承载无限场景
- **自然生发** — 拖拽即创造，所见即所得，无需编码亦能成事
- **和谐共生** — 组件互为依存，布局自洽平衡

## 规划功能

- 可视化画布编辑器（拖拽、缩放、对齐）
- 物料组件市场
- 模板管理与版本控制
- 多端预览与导出（打印、PDF、图片）
- 协作编辑

## 开发

```bash
# 从 monorepo 根目录启动
pnpm --filter banyan dev
```
